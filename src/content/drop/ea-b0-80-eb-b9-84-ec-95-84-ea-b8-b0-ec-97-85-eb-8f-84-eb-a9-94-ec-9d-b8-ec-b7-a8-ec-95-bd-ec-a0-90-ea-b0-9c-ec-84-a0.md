*   작업사항
    1.  메인 개편
    2.  푸터 수정
    3.  탑메뉴 수정
    4.  개인정보 취급방침 추가
    5.  약관 추가
    6.  개인정보 수정 페이지 수정
    7.  회원가입 페이지 수정
    8.  로그인 페이지 추가

    5.  개발서버와 실 서버의 환경차이로 인한 초기 셋팅 시간 지연
    6.  기존 CSS와 주소검색 CSS와의 충돌
