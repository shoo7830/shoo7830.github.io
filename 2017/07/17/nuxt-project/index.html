Nuxt/Yarn을 활용하여 개편 소개 페이지 만들기

Jul 17, 2017

올해 초 부터 Vue.js에 관심이 생겨 개인적으로 학습하고 있었다.
2017 1사분기를 회고하는 글에 이런 내용을 적었었다.

“회사 개발자 세미나에서도 해당 주제로 발표할 예정이며, 가능하다면 외부 밋업에서도 발표하고 싶다.”

회사에서 발표한 Vue.js와 Firebase를 활용기를 Vuetiful Korea 1회 모임에서 발표하였다.
이 날 많은 사람들 앞에서 발표한 것도 좋은 경험이였지만 Nuxt라는 범용 SSR 프레임워크를 알게 되었다.
언젠가 우리 서비스에 도입해야겠다고 생각했다. 내가 맡고 있는 하이웍스의 서비스 중 전자결재가 개편되었는데
이전 전자결재와 많은 부분이 달라 해당 내용을 소개하는 페이지가 필요했다.

단순하게 html과 css, jQuery를 써서 작업해도 되지만 Nuxt를 이용해서 만들어 보기로 했다.
무엇보다도 이렇게라도 써보지 않으면 앞으로도 못쓸것 같았다.

하이웍스 전자 결재 개편 소개

결론을 먼저 말하자면 일단 만들고 모르는 건 찾아보자라고 해서 그랬는지 심한 삽질의 연속이였다.

페이지 구조 구상

  • index
    • 반복되는 부분을 v-for를 써서 처리
  • sub page
    • layout을 만들어서 모든 subpage는 동일한 레이아웃 사용

필요한 개발 도구 설치

  1. Yarn

  2. Nuxt

개발하면서 얻은 팁

시행착오

  • CodIgniter는 my-procect같은 명명을 지원하지 않는다. 백엔드 개발자가 controller에서 해당 명명이 가능하도록 설정해주었다. 이 부분은 미리 백엔드 개발자와 논의했으면 불필요했을 부분이였다.
  • 정적인 파일로 생성해서 올리면 될 줄 알았는데 그게 아니였다. 요구 분석이 부족했다.
  • 다 만들고 나서 중첩된 라우터에 대한 부분을 확인하게 되었다. 이걸 진작 알았으면 삽질은 덜 했을텐데.
  • webpack의 동작원리를 모르니 이것 역시 삽질이
